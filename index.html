Phittawat

ช่วงที่ผมเรียนอยู่ ม.ต้น ผมได้มีโอกาสได้เข้าไปรู้จักครูที่อยู่กลุ่มสาระคอมพิวเตอร์ ในเวลาช่วงเย็นของทุกๆวัน ผมไปที่ห้องพักครู เพื่อไปหาควมรู้เกี่ยวกับคอมพิวเตอร์ ผมจึงมีโอกาสเรียนเสริมด้านการเขียนเว็บเบื้องต้น คือ Html และ CSS ทำให้ผมได้รู้ตัวเองว่าผมมีความชื่นชอบ และสนุกไปกับการเขียนโปรแกรม ผมจึงตัดสินใจที่จะเรียนรู้และศึกษาการเขียนโปรแกรมต่าง ๆ นอกเวลาเรียนเช่น python, C , Html, CSS , JavaScript และรวมไปถึงการประกอบคอมพิวเตอร์ และผมได้เข้าร่วมการแข่งขันต่าง ๆ ที่ใช้ทักษะทางคอมพิวเตอร์ เช่นการแข่งขันศิลปหัตถกรรมนักเรียนเกี่ยวกับการสร้าง WebApplication ที่ใช้ความรู้ด้านการเขียนเว็บไซต์ และการเชื่อมต่อกับฐานข้อมูล การแข่งขันสร้าง Infographic ที่ใช้ความรู้ทางศิลปะมาออกแบบ ทำให้ผมมั่นใจว่าชอบด้านคอมพิวเตอร์ ซึ่งนั่นทำให้ตัวผมอยากเข้าศึกษาต่อในคณะวิศวกรรมศาสตร์ สาขาวิศวกรรมคอมพิวเตอร์ ที่สถาบันเทคโนโลยีพระจอมเกล้าคุณทหารลาดกระบัง ซึ่งเป็นสถาบันที่มีชื่อเสียงทางด้านวิศวกรรม ในอนาคตผม หวังว่าจะได้ทำงานเกี่ยวกับการพัฒนา AI หรือปัญญาประดิษฐ์ที่สามารถทำให้เกิดประโยชน์แก่ทุกคนได้
